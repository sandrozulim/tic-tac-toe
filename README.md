# Tic-Tac-Toe

### Quickstart

- clone repo `https://github.com/sandrozulim/tic-tac-toe`
- cd tic-tac-toe
- `cp .env.template .env`
- yarn
- yarn dev

### Technology Stack

- React 18+
- TypeScript
- React Router
- React Query (TanStack Query)
- Axios
- Tailwind CSS for styling

### Project Description

A React + TypeScript web application for playing Tic-Tac-Toe against other players, using a REST API for backend game management. React Query is used for data fetching and state synchronization.
